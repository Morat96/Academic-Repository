Hierarchy of project folders:

Build -> bin: Contain executable and other useful files: more details in README.txt in that folder.

cpp_code: Contain the c++ code (main.cpp and the class PlateDetector).

PROJECT.pdf: Report of the project.

ARIAL.csv.zip: Zip of the dataset for the neural network training (unzip the file for train the model).

training_model.py: Python program for train the neural network for classify license plate digits, return the model described by files: neural_network.json (structure of the network) and neural_network.h5 (weights).
